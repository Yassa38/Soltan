# بوت المواعيد 📅

بوت تيليجرام لإدارة المواعيد مع تذكيرات تلقائية.

## هيكل المشروع

```
appointments_bot/
├── main.py          # نقطة الدخول – تسجيل الهاندلرز وتشغيل البوت
├── config.py        # الإعدادات والثوابت (التوكن، المنطقة الزمنية، حالات المحادثة)
├── database.py      # كل عمليات قاعدة البيانات (CRUD)
├── handlers.py      # منطق الهاندلرز (الأوامر، المحادثات، الأزرار)
├── keyboards.py     # بناء لوحات المفاتيح التفاعلية
├── utils.py         # دوال مساعدة (التواريخ، التنسيق، العلامات)
└── requirements.txt
```

## التثبيت

```bash
pip install -r requirements.txt
```

## التشغيل

```bash
export TELEGRAM_BOT_TOKEN="توكن_البوت_هنا"
export BOT_TIMEZONE="Asia/Riyadh"   # اختياري
export BOT_DB_PATH="appointments.db" # اختياري

python main.py
```

## الأوامر

| الأمر | الوصف |
|-------|-------|
| `/start` | القائمة الرئيسية |
| `/add` | إضافة موعد جديد |
| `/list` | عرض المواعيد القادمة |
| `/delete <id>` | حذف موعد برقمه |
| `/cancel` | إلغاء العملية الحالية |
